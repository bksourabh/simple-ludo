Classic Board Game Fun

Version 1.1

Features

Room Code Multiplayer NEW

Create a game, get a unique code, and share it with friends. No Game Center friend list needed — just enter the code and play.

Mid-Game Late Join NEW

Friends can join even after the game starts. AI players are seamlessly replaced with human players, with host approval.

Classic Ludo Gameplay

Faithful to the original rules with smooth animations. Roll the dice, race your tokens, and be the first to get all four home.

Play Your Way

Any combination of Human and Computer players. Solo against AI, local pass-and-play, or online with friends.

Beautiful Graphics

Vibrant, hand-crafted game board with smooth SpriteKit animations. Tokens glide, dice roll with style, and victories celebrate.

Family Friendly

No ads. No in-app purchases. No data collection. Just pure, classic board game fun for the whole family.

What's New in v1.1

  • Room Code Multiplayer — create a game, get a 6-character code, share via Messages or clipboard
  • Host Colour Selection — choose your colour before the game starts
  • Auto-Start — game starts automatically when all 4 human players join
  • AI Backfill — start early with remaining slots filled by AI
  • Mid-Game Late Join — friends join mid-game, replacing AI players with host approval
  • Guest Online Access — guest users can now play online (Game Center sign-in prompted on demand)
  • UI Improvements — improved menu layout, refined colour picker, better button alignment
